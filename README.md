# Portfolio Android App

Bu loyiha Android Studio uchun shaxsiy portfolio ilovasi namunasidir.

## Sahifalar

1. Home (Bosh sahifa)
- Talaba rasmi (placeholder)
- Ism
- Qisqacha tanishtiruv

2. About (Haqimda)
- Ism
- O'qish joyi
- Organgan texnologiyalar

3. Projects (Loyihalar)
- Loyiha nomi
- Ishlatilgan texnologiyalar
- Linkni ochish tugmasi

4. Contact (Kontaktlar)
- Email
- Telefon
- Ijtimoiy tarmoqlar linklari

## Texnologiyalar

- Android Studio
- Kotlin
- XML
- ViewPager2 + TabLayout
- RecyclerView

## Ishga tushirish

1. Android Studio orqali `android` papkani oching.
2. Gradle sync tugashini kuting.
3. Emulator yoki fizik qurilma tanlang.
4. Run tugmasini bosing.
